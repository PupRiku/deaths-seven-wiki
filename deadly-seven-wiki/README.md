# The Deadly Seven — DM Wiki

A local Next.js web application for running The Deadly Seven D&D 5e campaign.

## Features

- 📖 Chapter navigation with full module content
- 👥 NPC quick reference (searchable)
- ⚔️ Initiative & encounter tracker with stat blocks
- 📝 Live session note capture
- ⬇️ Export notes as .md or .txt for sharing

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Stack

- Next.js 14 + TypeScript
- Tailwind CSS
- libSQL (SQLite) for session data persistence
- Cinzel / EB Garamond typography
